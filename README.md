# イライラ棒featこうかとん

## 実行環境の必要条件
* python >= 3.10
* pygame >= 2.1

## ゲームの概要
* 主人公キャラクターこうかとんを操作し、棒をぶつけない様にクリアする

## ゲームの遊び方
* 矢印キーでこうかとんを操作する。
* ぶつかったらゲームオーバー

## ゲームの実装
### 共通基本機能
* 背景画像と主人公キャラクターの描画

### 分担追加機能
* 壁を生成　役割：早川　航
* 障害物（壁みたいなもの）役割：佐藤　光暉
* 障害物（ボムみたいなもの）役割：坂本　直輝
* ステージ機能　役割：大橋　諒平
* スコア、タイム　役割：小澤　虎ノ介
